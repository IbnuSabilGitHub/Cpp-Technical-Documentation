C++ Technical Documentation

Introduction

C++ is a statically typed, compiled, general-purpose programming language designed as an extension of the C language by Bjarne Stroustrup in 1979. It supports multiple programming paradigms, such as procedural, object-oriented, and generic programming, making it highly versatile for various applications like system/software development, game development, embedded systems, and real-time simulations.

C++ provides low-level memory manipulation features while also supporting high-level abstraction through object-oriented techniques. It is widely used for performance-critical applications where efficiency and speed are of utmost importance.

Syntax Overview

1. Basic Structure

    
                            
    
                                cpp
                                
    
                                    
                                
    
                            
    
                            
    #include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
  • #include <iostream> This preprocessor directive includes the standard input/output stream library.
  • int main(): This is the main function where program execution begins.
  • std::cout: This is used to print output to the console.
  • std::endl: Inserts a newline and flushes the output buffer.
  • return 0: Indicates successful program termination.

2. Data Types

  • Primitive Data Types:

    int: Integer type (e.g., int x = 10; ).

    float, double: Floating-point types (e.g., float pi = 3.14f; ).

    char: Character type (e.g., char letter = 'A'; ).

    bool: Boolean type (e.g., bool isTrue = true; ).

    string: For handling strings, typically with the standard library (e.g., std::string name = "C++"; ).

  • Derived Data Types:

    Array: Collection of elements of the same type (e.g., int arr[5]; ).

    Pointer: Stores memory address of another variable(e.g., int* ptr = &a; ).

  • User-Defined Data Type:

    struct, class, enum union: Used to define custom data types.

3. Variables and Constants

Variables are used to store data in memory, while constants are immutable values that cannot change during program execution. 

                            

                                cpp
                                

                                    
                                

                            

                            
int age = 25;           // Variable
const double PI = 3.14; // Constant

4. Operators

Operators are used to perform operations on variables and values. C++ supports several types of operators:

  • Arithmetic Operators: +, -, *, /, %,
  • Relational Operators: ==, !=, >, <, >=, <=,
  • Logical Operators: &&, ||, !,
  • Assignment Operators: =, +=, -=, *=, /=, %=,

5. Control Flow Statements

    Conditional statements are used to perform different actions based on conditions.

    • For Loop:

Object Oriented Programming

1. Classes and Objects

    
                                
    
                                    cpp
                                    
    
                                        
                                    
    
                                
    
                                
    class Car {
public:
    std::string brand;
    int year;

    void honk() {
        std::cout << "Honk! Honk!" << std::endl;
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2020;
    myCar.honk();
}
  • Class:blueprint for objects, defining properties (e.g., brand , year ) and methods(e.g., hook() ).
  • Object: An instance of a class (e.g., mycar ).

2. Inheritance

    
                                
    
                                    cpp
                                    
    
                                        
                                    
    
                                
    
                                
    class Vehicle {
public:
    int wheels;
};

class Car : public Vehicle {
public:
    std::string brand;
};
  • Base Class: The class being inherited from ( Vehicle ).
  • Derived Class: The class that inherits ( mycar ).

3. Polymorphism

    
                                
    
                                    cpp
                                    
    
                                        
                                    
    
                                
    
                                
    class Animal {
public:
    virtual void sound() {
        std::cout << "Some sound" << std::endl;
    }
};

class Dog : public Animal {
public:
    void sound() override {
        std::cout << "Woof!" << std::endl;
    }
};

int main() {
    Animal* a = new Dog();
    a->sound();
}
  • Virtual Function: A function that can be overridden in a derived class.
  • Override: Replacing a base class method with a derived class method..

4. Encapsulation

    
                                
    
                                    cpp
                                    
    
                                        
                                    
    
                                
    
                                
    class Employee {
private:
    int salary;

public:
    void setSalary(int s) {
        salary = s;
    }
    
    int getSalary() {
        return salary;
    }
};
  • Private Members: Members that cannot be accessed outside the class.
  • Override: Methods to access and modify private members.
